This directory contains source or raw data for analysis.

Files
-----
*.jpg - files used as icons in plots.
camera configs.csv - describes the different cameras used to record trajectories.
TrajectoryData.rds - cached trajectory data.

Directories
-----------
.cache - cached trajectory autocorrelation data, create as needed
outlines - contains outline images of lots of specimens. Details are in README.txt in the outlines directory.
sampleITcache - contains information about samples.
videos - contains trajectory videos, meta data and extracted tracks. See README.txt in videos/lab for more info.
